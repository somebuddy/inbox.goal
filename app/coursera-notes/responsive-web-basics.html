  • It should display correctly if the person viewing it has placed all the files you have supplied in a folder on their machine

    My comment: I hope it works on your PC in your browser. This page was tested with latest versions Chrome, Firefox, Edge. Other browsers and versions were not tested.

  • It should use at least one of each of HTML img, h1 and div tags (and more if needed)
    My comment:
    • img: for github link I used official logo image
    • h1: page header, largest text in h1-element
    • div: here are a lot of such elements
  • It should use CSS to set the style of the text and other parts of the design as appropriate, in at least 3 ways.
    My comment: here's many styles (almost all page, except additional elements: modals, tooltips, circular progress bar etc. - for those I used bootstrap and jquery knob).

    But, I used Sass and gulp to build final css file, so it little ugly formatted. Visit github repository to view source.

  • It should be responsive - it should change the layout if the screen size changes
    My comment:
    • Navigation bar: collapsed on small devices (using bootstrap)
    • Day statistics: history plot is hidden on small devices
    • Goal widget: as full width row on small devices and as card on tables and desktops
  • The user should be able to change the page in some way by interacting with a GUI element such as a button.
    My comment:
    • Current day widget: You could change current day left/right chevron to decrease/increase displayed day
    • History chart: You could change current day by clicking some bar
    • Add goal button: You could add new random goal. Creating user goal is not ready yet.
  • You can also upload additional files such as jQuery or Bootstrap if needed.
    My comment:
    • JQuery: I used JQuery for elements manipulation in script.
    • Bootstap: I used Bootstrap for some elements: button, navbar, icons, tooltips, popovers, modals etc.
    • MomentJS: for date display and calculations in script (Moment.js).
    • JQuery Knob: I used JQuery Knob progress bar on the statistics bar.
